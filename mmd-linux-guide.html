How to run MMD on Linux (64-bit)

TH คู่มือการลง MMD สำหรับ Ubuntu 24.04.3 LTS, ZorinOS 18 และ Linux Mint 22.3+

EN Comprehensive MMD installation guide for modern Ubuntu-based systems.

JP 最新のUbuntu/ZorinOS環境でMMD（64bit）をセットアップするための詳細ガイド。

⚠️ CRITICAL NOTICE / 重要事項 / ข้อควรระวัง

TH: เวอร์ชั่น 64-bit ไม่รองรับการแตกไฟล์ขณะเรนเดอร์ (Video Compression) แนะนำให้ใช้ Avi Raw หรือ MJPEG เท่านั้น!

EN: 64-bit version lacks internal video compression support. Export using Avi Raw or MJPEG only.

JP: 64bit版はビデオ圧縮に対応していません。Avi RawまたはMJPEG形式のみを使用してください。

01

Initial Environment Setup

TH ติดตั้ง Wine (Staging) และ Wintricks เวอร์ชั่นล่าสุดเพื่อรองรับแอปพลิเคชัน Windows

EN Install the latest Wine (Staging) and Wintricks for Windows app compatibility.

JP 最新のWine (Staging) と Wintricks をシステムに導入します。

sudo dpkg --add-architecture i386
sudo mkdir -pm 755 /etc/apt/keyrings
sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/noble/winehq-noble.sources
sudo apt update && sudo apt install --install-recommends winehq-staging winetricks -y
02

Download Automation Script

TH ดาวน์โหลดสคริปต์ติดตั้งอัตโนมัติ installmmd64.sh

EN Download the installmmd64.sh automation script.

JP 自動インストールスクリプト installmmd64.sh を取得します。

DOWNLOAD installmmd64.sh
03

Execute & Lutris Configuration

1. Running the Script / スクリプトの実行

TH เปิด Terminal ในโฟลเดอร์ที่ดาวน์โหลดสคริปต์ไว้ แล้วรันคำสั่งด้านล่าง

EN Open your terminal in the script directory and execute these commands.

JP スクリプトを保存したフォルダでターミナルを開き、コマンドを実行してください。

chmod +x installmmd64.sh
./installmmd64.sh

2. Lutris & Wine Configuration (HACKER STYLE)

  • TH ไปที่เมนู Configure > Runner options ใน Lutris
  • EN Go to Configure > Runner options in Lutris.
  • JP Lutrisの Configure > Runner options に移動します。

  • 🚨 CRITICAL STEP / 重要 / ขั้นตอนสำคัญ:

    TH ในช่อง Wine version **ห้ามเลือก (Default)** ให้เลือกเป็น Wine (Staging) 11.x - HQ หรือเวอร์ชั่นที่ใหม่กว่าเท่านั้น!

    EN Under Wine version, **DO NOT select (Default)**. You must choose Wine (Staging) 11.x - HQ or a newer version.

    JP Wine version の項目で **(Default) を選択しないでください**。必ず Wine (Staging) 11.x - HQ またはそれ以降のバージョンを選択してください。

  • TH ตั้งค่า Wine Prefix ไปที่ ~/.WineMMD64 และเลือกโฟลเดอร์หลัก MikuMikuDance_V932x64
  • EN Set Wine prefix to ~/.WineMMD64 and point to MikuMikuDance_V932x64 folder.
04

MME64 & FFmpeg Engine

TH ลง MME64: ก๊อปปี้ไฟล์ d3d9.dll ไปวางในโฟลเดอร์ MMD.exe และติดตั้ง FFmpeg เพื่อแปลงไฟล์

EN Install MME64: Copy d3d9.dll to the MMD.exe directory and install FFmpeg.

JP MME64の導入: d3d9.dll を MMD.exe と同じディレクトリに配置し、FFmpegをインストールします。

sudo apt install ffmpeg -y

Conversion Command / 変換コマンド:

ffmpeg -i input.avi -vcodec libx264 -crf 18 output.mp4
