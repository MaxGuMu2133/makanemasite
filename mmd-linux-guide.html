MMD & PMX Editor Linux Setup

TH คู่มือการลง MMD และ PMX Editor สำหรับ Ubuntu / ZorinOS แบบละเอียด

EN Detailed MMD & PMX Editor installation for Ubuntu-based Linux.

JP LinuxでMMDとPMX Editorをセットアップするための完全ガイド。

REQUIRED SOFTWARE

MikuMikuDance V932

64-Bit (ZIP) 32-Bit (ZIP)

PMXEditor 0254f

English Ver. (DeviantArt)

MikuMikuEffect V037

64-Bit (MME) 32-Bit (MME)

** Note: External download links from LearnMMD & DeviantArt **

01

Initial Environment Setup

sudo dpkg --add-architecture i386
sudo mkdir -pm 755 /etc/apt/keyrings
sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/noble/winehq-noble.sources
sudo apt update && sudo apt install --install-recommends winehq-staging winetricks -y
02

Linux Automation Scripts

DOWNLOAD: installmmd64.sh DOWNLOAD: installPMXEditor.sh

(TH) โหลดสคริปต์ทั้งคู่ไปวางในโฟลเดอร์เดียวกับโปรแกรมที่โหลดมาด้านบน / (JP) 同じフォルダに保存してください。

03

Execute & Lutris Configuration

chmod +x *.sh
./installmmd64.sh
./installPMXEditor.sh

Path Mapping / パス設定:

[ MikuMikuDance ]

> Executable: MikuMikuDance.exe

> Prefix: .WineMMD64

[ PMX Editor ]

> Executable: PMXEditor.exe

> Prefix: .WinePMX

Runner Options:

TH เลือก Wine version เป็น WineHQ Staging 11.4 เท่านั้น! ห้ามเลือก (Default)

04

MikuMikuEffect (MME) Injection

TH ก๊อปปี้ไฟล์ d3d9.dll และ MMEffect.dll จากไฟล์ที่โหลดมาไปใส่ใน Folder MikuMikuDance

JP d3d9.dll と MMEffect.dll を MikuMikuDance フォルダにコピーします。

05

FFmpeg Video Engine

sudo apt install ffmpeg -y

Conversion: ffmpeg -i input.avi -vcodec libx264 -crf 18 output.mp4
