How to run MMD on Linux (64-bit)

TH คู่มือการลง MMD สำหรับ Ubuntu 24.04.3 LTS, ZorinOS 18 และ Linux Mint 22.3+

EN MMD Installation guide for Ubuntu-based Linux distros.

JP LinuxでMMD（64bit）を動作させる方法。Ubuntu/ZorinOS対応。

⚠️ IMPORTANT / 重要 / ข้อควรระวัง

(TH) MMD 64-bit ไม่รองรับการแตกไฟล์ขณะเรนเดอร์ แนะนำให้ใช้ Avi Raw หรือ MJPEG เท่านั้น!
(EN) 64-bit version does not support compressed AVI. Use Avi Raw or MJPEG only.
(JP) 64bit版はAVI圧縮に対応していません。Avi RawまたはMJPEGを使用してください。

STEP 1: Install Wine (Staging) & Wintricks

ติดตั้ง Wine เวอร์ชั่นล่าสุดเพื่อให้รองรับโปรแกรม Windows

sudo dpkg --add-architecture i386
sudo mkdir -pm 755 /etc/apt/keyrings
sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/noble/winehq-noble.sources
sudo apt update && sudo apt install --install-recommends winehq-staging winetricks -y

STEP 2: Download Automation Script

คลิกปุ่มด้านล่างเพื่อโหลดไฟล์สคริปต์ติดตั้งอัตโนมัติ

DOWNLOAD installmmd64.sh

STEP 3: Lutris & MMD Configuration

  • TH: ติดตั้ง Lutris และสร้าง Prefix ใหม่ที่ ~/.WineMMD64
    EN: Open Lutris, create a new Wine prefix at ~/.WineMMD64
  • TH: เลือกไฟล์ MMD ที่โหลดมา (เช่น MikuMikuDance_V932x64)
    EN: Select MMD main folder MikuMikuDance_V932x64

STEP 4: MME & Video Conversion (FFmpeg)

ลง MME64 โดยการก๊อปปี้ไฟล์ d3d9.dll ไปวางที่เดียวกับ MMD.exe

sudo apt install ffmpeg -y

Use command: ffmpeg -i input.avi -vcodec libx264 -crf 18 output.mp4
