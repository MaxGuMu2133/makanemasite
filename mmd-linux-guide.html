MMD & PMX Editor Linux Setup

TH คู่มือการลง MMD และ PMX Editor สำหรับ Ubuntu 24.04.3 LTS, ZorinOS 18 และ Linux Mint 22.3+

EN Full installation guide for MMD & PMX Editor on modern Ubuntu-based systems.

JP Linux（ZorinOS/Ubuntu）環境でMMDとPMX Editorをセットアップする完全ガイド。

⚠️ CRITICAL NOTICE / 重要事項 / ข้อควรระวัง

(TH) MMD 64-bit ไม่รองรับการแตกไฟล์ขณะเรนเดอร์ แนะนำให้ใช้ Avi Raw หรือ MJPEG เท่านั้น!

(JP) 64bit版はAVI圧縮に対応していません。Avi RawまたはMJPEGを使用してください。

01

Initial Environment Setup

sudo dpkg --add-architecture i386
sudo mkdir -pm 755 /etc/apt/keyrings
sudo wget -O /etc/apt/keyrings/winehq-archive.key https://dl.winehq.org/wine-builds/winehq.key
sudo wget -NP /etc/apt/sources.list.d/ https://dl.winehq.org/wine-builds/ubuntu/dists/noble/winehq-noble.sources
sudo apt update && sudo apt install --install-recommends winehq-staging winetricks -y
02

MMD 64-bit Installation

TH ดาวน์โหลดและรันสคริปต์ installmmd64.sh

DOWNLOAD installmmd64.sh
chmod +x installmmd64.sh && ./installmmd64.sh

[Lutris Config]

1. Executable: MikuMikuDance.exe

2. Working Directory: Folder MikuMikuDance

3. Wine Prefix: .WineMMD64

4. Wine Version: WineHQ Staging 11.4 (ห้ามเลือก Default!)

03

PMX Editor Installation

TH ดาวน์โหลดสคริปต์ installPMXEditor.sh เพื่อเตรียมสภาพแวดล้อมสำหรับ PMX Editor

JP PMX Editor用のインストールスクリプト installPMXEditor.sh をダウンロードします。

DOWNLOAD installPMXEditor.sh
chmod +x installPMXEditor.sh
./installPMXEditor.sh
04

Lutris Config for PMX

Lutris Settings / 設定:

> [Executable]: PMXEditor.exe

> [Working directory]: โฟลเดอร์หลักของ PMXEditor

> [Wine prefix]: .WinePMX (แตกมาจากสคริปต์)

🚨 Compatibility / 互換性:

TH ในหน้า Configure > Runner options เลือก Wine version เป็น WineHQ Staging 11.4 เท่านั้น! **ห้ามใช้ตัว (Default)**

JP Runner optionsの Wine version で必ず WineHQ Staging 11.4 を選択してください。**(Default) は使用禁止**。

05

FFmpeg Video Engine

sudo apt install ffmpeg -y

Convert AVI to MP4: ffmpeg -i input.avi -vcodec libx264 -crf 18 output.mp4
